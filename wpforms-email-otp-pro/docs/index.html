1. Installation

1 Upload the Plugin

  1. Go to WordPress Admin → Plugins → Add New
  2. Click Upload Plugin at the top
  3. Select the wpforms-email-otp-pro.zip file
  4. Click Install Now

2 Activate the Plugin

After installation is complete, click Activate Plugin.

2. Configuration

1 Access Plugin Settings

Navigate to WPForms → Settings → Email OTP tab.

2 Configure Basic Settings

  1. Enable SMTP: Turn this on to use Gmail for sending OTP emails
  2. Sender Email: Enter your Gmail address (e.g., yourname@gmail.com)
  3. App Password: Enter your Gmail App Password (see Gmail Setup)

3 Form Configuration

  1. WPForms Form ID: Select which form should use OTP verification
  2. Email Field Label: Enter the exact label of your email field (case sensitive)
  3. OTP Length: Choose between 4, 6, or 8 digit codes
  4. OTP Expiry: Set how many minutes the code should remain valid

3. Gmail App Password Setup

1 Enable 2-Step Verification

  1. Go to your Google Account
  2. Navigate to Security → 2-Step Verification
  3. Follow the steps to enable 2-Step Verification

2 Generate App Password

  1. In your Google Account, go to Security → App Passwords
  2. Select Mail as the app
  3. Select Other (Custom name) and enter "WPForms OTP"
  4. Click Generate
  5. Copy the 16-character password that appears
Important: This password will only be shown once. Store it securely.

3 Enter App Password in Plugin

Paste the generated password in the App Password field in the plugin settings.

4. Form Setup

1 Create/Edit Your Form

Ensure your form has an email field with the exact label you specified in settings.

2 How It Works for Users

  1. User enters their email address
  2. Clicks "Send OTP" button
  3. Receives email with verification code
  4. Enters code in the form
  5. Clicks "Verify OTP"
  6. If correct, the form submit button is enabled

5. Custom Email Templates

1 Create an Elementor Template

  1. Go to Templates → Saved Templates
  2. Click Add New and select "Email"
  3. Design your template using Elementor
  4. Use these placeholders where needed:
    • {otp} - The verification code
    • {site} - Your website name
    • {email} - Recipient's email address
  5. Save the template

2 Assign Template in Plugin

  1. Note the template ID (visible in URL or template list)
  2. Go to WPForms → Settings → Email OTP
  3. Enter the template ID in the "Elementor Template ID" field
  4. Save changes

6. Troubleshooting

Emails Not Sending

  • Verify your Gmail App Password is correct
  • Check that 2-Step Verification is enabled in your Google Account
  • Test with a different SMTP plugin to isolate the issue

OTP Verification Not Working

  • Ensure the email field label matches exactly (case sensitive)
  • Check that the correct form is selected in settings
  • Verify your server's time is synchronized correctly

Elementor Template Not Loading

  • Ensure Elementor is installed and activated
  • Verify the template ID is correct
  • Check that the template type is "Email"
